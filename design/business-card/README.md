# Peco&Kb 名刺デザイン

標準的な日本の名刺サイズ（91 × 55 mm）で制作した両面SVGです。

- `business-card-front.svg`: 表面
- `business-card-back.svg`: 裏面
- `business-card-preview.svg`: 両面を並べた確認用プレビュー

## デザイン方針

- Webサイトのブランドカラー（濃紺 `#07101f`、ブルー `#143c73`、オレンジ `#ff6533`）を継承
- 表面はAI・ネットワークを想起させるノード表現
- 裏面は読みやすさを優先した白ベースに、`img/KB.jpg`の白黒ポートレートを配置
- 氏名、会社所在地、法人番号はWebサイト掲載内容を反映

## 入稿前に確認する項目

- 電話番号、メールアドレス、WebサイトURLは元サイトに有効な情報がなかったため未掲載
- 印刷会社の指定に応じて、文字のアウトライン化、3 mm塗り足し、CMYK変換を行う
- 氏名表記はサイトの英字表記 `Yoshimi Tsujimura` と文字化けデータから `辻村 圭美` と復元
